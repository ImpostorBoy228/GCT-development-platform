
### Инструкции по запуску Codex Omni

1. Установите Node.js и Python 3.8+.
2. Установите зависимости фронтенда:
   ```bash
   cd frontend
   npm install
   ```
3. Запустите фронтенд:
   ```bash
   npm start
   ```
4. Установите зависимости бэкенда:
   ```bash
   cd backend
   pip install -r requirements.txt
   ```
5. Запустите бэкенд:
   ```bash
   node server.js
   ```
6. Откройте в браузере `http://localhost:3000`.

Приятной разработки, тёмный повелитель!
